Description
Welcome PM Panel for sending new registered members a welcome private message after they have registered and logged into a site for the first time.
  • Welcome Panel can be enabled or disabled in the infusion admin.
  • Welcome Message Subject and message fully customisable in the infusion admin.
  • Admin Preview in the infusion admin
  • No modifications of PHP-Fusion core files required.
Install
  1. Upload all files and folders in the main /files/ folder.
  2. Go to Admin >>> System  >>> Infusions
  3. Select welcome pm panel  from the list and infuse.
  4. Done, panel is enabled automatically when infusing and deleted when defusing.

 

Usage
Go to infusions/welcome_pm_panel_admin.php to enable Welcome PM, set welcome message subject, and message.
